# MKV Subtitle Extractor

A local web application to extract subtitles from MKV video files.

## Features

- Upload MKV files
- Extract multiple subtitle tracks
- Convert ASS subtitles to SRT format
- Fast local processing
- Download extracted subtitles

## Requirements

- Windows (64-bit)
- No additional software needed - everything is bundled

## Installation

No installation required! The app is a standalone executable.

## Usage

1. Download the `app.exe` file
2. Double-click `app.exe` to run it
3. Your default web browser will automatically open to the app
4. Upload an MKV file using the form
5. Download the extracted subtitle files

## How it works

- The app uses bundled MKVToolNix to analyze and extract subtitle tracks from MKV files
- ASS (Advanced SubStation Alpha) subtitles are automatically converted to SRT format
- UTF-8 text subtitles are extracted as-is
- All processing happens locally on your computer for speed and privacy

## Supported Formats

- Input: MKV files (up to 10GB)
- Output: ASS and SRT subtitle files
