# MKV Media Processor

A local web application to extract subtitles, audio, and convert MKV video files.

## Features

- Upload MKV files
- Extract multiple subtitle tracks
- Extract audio tracks
- Convert video to different formats (MP4, AVI, MKV)
- Change video resolution (480p, 720p, 1080p)
- Translate subtitles using free translation service (Google Translate)
- Convert ASS subtitles to SRT format
- Fast local processing
- Download processed files

## Requirements

- Windows (64-bit)
- No additional software needed - everything is bundled

## Installation

No installation required! The app is a standalone executable with all tools bundled.

## Usage

1. Download the `MKV_Media_Processor.zip` file
2. Extract the zip file (you'll get just one exe file and README)
3. Double-click `MKV Media Processor.exe` to run it
4. Your default web browser will automatically open to the app
5. Upload an MKV file using the form
6. Select the operations you want (extract subtitles, extract audio, convert video)
7. Optionally enable AI translation for subtitles and select target language
8. Choose output format and resolution if converting
9. Click "Process File"
10. Download the processed files

## How it works

- The app uses bundled MKVToolNix and FFmpeg to analyze and process MKV files
- ASS subtitles are automatically converted to SRT format
- Audio tracks are extracted as MP3
- Video conversion supports resolution scaling
- All processing happens locally on your computer for speed and privacy

## Supported Formats

- Input: MKV files (up to 10GB)
- Output: ASS, SRT subtitle files; MP3 audio; MP4/AVI/MKV video files
- AI Translation: Multiple languages supported via free Google Translate service
